IbexHelper:
-------------

Helper functions for [Ibex Package](https://github.com/bmatejek/ibex) for the manipulation of skeleton and its simplified graph.

Install
===
```
cd ibex/skeletionization/
python setup.py build_ext --inplace
cd ../../ibex/transforms/
python setup.py build_ext --inplace
```
